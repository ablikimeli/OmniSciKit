# OmniSciKit 0.1.0

## Initial Release

### New Features
- Comprehensive toolkit for scientific research with 6 main modules
- Over 150 functions covering bioinformatics, Mendelian randomization, machine learning, epidemiology, statistics, and sample size calculation
- Built-in demo datasets (omni_demo, omni_epi, omni_ml)
- Complete documentation with examples
- 7 vignettes for detailed tutorials
- GitHub Actions CI/CD workflow
- Unit tests for core functions

### Modules
1. **Bioinformatics** (omics_*) - Differential expression, pathway enrichment, data download
2. **Mendelian Randomization** (mr_*) - Causal inference with genetic instruments
3. **Machine Learning** (ml_*) - Classification, regression, model evaluation
4. **Epidemiology** (epi_*) - Disease modeling, outbreak analysis
5. **Statistics** (stats_*) - Hypothesis testing, regression, survival analysis
6. **Sample Size** (ss_*) - Power analysis, study design

### Utilities
- Environment management (check_deps, install_omni)
- Data connection (omni_connect_db, omni_query_db)
- Reporting (omni_report, omni_summary)
- Visualization themes (omni_theme, omni_palette)
- Data utilities (omni_save_data, omni_load_data)

### Documentation
- Complete function documentation with examples
- 7 vignettes covering all modules
- README with quick start guide
- CRAN-compliant package structure

### Testing
- Unit tests for core functions
- Test coverage for statistics and sample size modules
- Continuous integration with GitHub Actions
